从C到Java值得注意的事情

本文面向学习过C语言后想入门学习Java基本语法的读者.(俺水平十分有限,仅提供入门参考!!)

因为我在学Java语法的时候,发现很多地方和C不能说是一模一样,但也是大差不差的,所以只需要我们只需要关注一些值得注意的特性(或是不同点)来学就可以了.

持续更新中……

数组

  1. java数组有内部成员 length 告诉你元素数量。

    （相当于C语言中的 sizeof(array)/sizeof(type of array) ）

  2. 元素个数可以是变量。

字符串

  1. 字符串操作：
    • 比较大小
    • 获得长度
    • 访问某个位置的字符
    • 取得子串（子集）
    • 寻找特定字符
  2. 所有的字符串都是不可变的，对它们的操作结果都是制造新的字符串出来。

对象

  1. 创建对象的时候发生了什么 && “new”是什么意思：

    以字符串变量为例：

    new String：创建一个String对象。

    ("a string")：用“a strong”初始化这个对象。

    String s = new String();：创建管理这个对象的变量s。
