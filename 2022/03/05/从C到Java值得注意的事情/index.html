从C到Java值得注意的事情

发表于 2022-03-05 更新于 2022-04-03 分类于 学习 阅读次数：

本文面向学习过C语言后想入门学习Java基本语法的读者.
(俺水平十分有限,仅提供入门参考!!)
因为我在学Java语法的时候,发现很多地方和C不能说是一模一样,但也是大差不差的,所以只需要我们只需要关注一些值得注意的特性(或是不同点)来学就可以了.

持续更新中……

变量与计算

  1. java变量一般不允许不初始化就使用，不能像c那样随便声明一个变量看它奇奇怪怪的默认值。

    例外情况（使用默认值）：

    • 变量是类的成员变量

    • 创建数组

  2. 相比c语言多了boolean类型，只有true和false两个值，相当于c里的1和0。

  3. 基本数据类型都有对应的包裹类型（使其对象化），可以像对象一样用”.“运算符使用函数。（如Integer.MAX_VALUE可以获取Integer类型的最大值）

    基本类型 包裹类型
    boolean Boolean
    char Character
    byte Byte
    short Short
    int Integer
    long Long
    float Float
    double Double

判断

  1. switch case:

    case除了支持int,char,short，还可以支持String(JDK7)、Lambda表达式(JDK12)。

  2. if()内只能用布尔表达式（c语言可以写if(1)，但java不行，因为”1“是int类型）。

循环

  1. 没有goto跳转（经典为了安全删删删~），但有标号语句跳转。

    标号语句有一些限制（没错，也是为了安全~）：

    • 只能跳出语句块。
    • 只能跳出break语句所在的外部语句块。

  2. 特别地，有增强for循环。

    在for()内：”变量“ + ”:” + “数组” 的写法叫做增强for循环。

    冒号“:”表示迭代，即不断从数组numArray中取出元素赋值给临时变量i，直到最后一个元素。

  3. 同判断一节，for、while的条件都必须是布尔表达式，c语言里无限循环while(1)在java里要写成while(true)。

数组

  1. 声明数组写法和c略有不同。

  2. java数组有内部成员 length 告诉你元素数量。

    （相当于C语言中的 sizeof(array)/sizeof(type of array) ）

  3. 定义数组时，元素个数可以是变量。

  4. 定义数组时，可以不为数组元素分配内存空间，之后再指明数组长度。

字符串

  1. 字符串对象自带操作：

    （用”.”运算符调用）

    • 比较大小
    • 获得长度
    • 访问某个位置的字符
    • 取得子串（子集）
    • 寻找特定字符
    • ······(等等)

  2. 所有的字符串都是不可变的，对它们的操作结果都是制造新的字符串出来。

函数/方法

  1. java，或者说面向对象语言里多了“方法（Method）“的概念，区别于”函数（Function）“

    啥是方法呢？可以理解为只属于类的函数，类内部的函数，就叫做方法。

    函数是大家的函数，方法是类的方法。

    一个东西，用对象调用的叫方法，直接调函数名的叫函数。

    String s = new String();：创建管理这个对象的变量s。
